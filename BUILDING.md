# PC
## Linux
You will need the following dependencies:
- SDL2
- SDL2_image
- SDL2_ttf
- OpenAL
- libvorbis
- gcc
- make

1. Run ``make pc``
2. Run ``make install`` (with root permissions)

After installing, libflixel++.so will be placed in /usr/lib/ and the neccessary headers will be placed in /usr/include/flixel++
